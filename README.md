# TS-Visualizer
<br />
Java based application for DVB Transport Stream analysis and visualization.<br />
Supports 188B-packet streams and .ts, .ts, .tp, .trp, .m2t, .m2ts, .mpg, .pcap and .vob file formats.<br />
